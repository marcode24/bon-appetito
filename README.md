### üì¶ Installation & Usage

```bash
# Clone this repository
git clone https://github.com/marcode24/bon-appetito

# Go into the repository
cd bon-appetito

# Install dependencies
npm install

# Run the app
ng serve
```

shut it down manually with `Ctrl-C` or `Cmd-C`.
